**Sections:** §10.3
**Issue:** Tests assert enum ordering uses declaration order (not lexicographic), but §10.3 never defines enum sort semantics.
**Proposed resolution:** Add a normative rule in §10.3: "Enum values sort by declaration order; descending reverses that order."
**Resolution:** §10.3 already specifies enum ordering; kept and relied on existing tests.
