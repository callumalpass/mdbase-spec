**Sections:** §14.3, §14.1, tests/level-1/collection-layout.yaml
**Issue:** Multiple Level 1 tests use `operation: query`, but querying is a Level 3 capability. The test format does not clarify whether `query` is allowed as infrastructure at lower levels or whether these tests implicitly require Level 3.
**Proposed resolution:** Document in §14.3.1 that `query` may be used as test infrastructure for low-level tests, or refactor those tests to use read/validate instead.
**Resolution:** §14.3.1 now defines a minimal query subset required for Level 1–2 conformance tests.
