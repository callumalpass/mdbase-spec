**Sections:** §12.2, §6.3–§6.4, §14.1
**Issue:** §12.2 step 3 says to "Evaluate match rules for all types" when determining types on read. Match rules are Level 2, but read is Level 1. This mirrors SN-075 for create and creates a cross-level dependency.
**Proposed resolution:** Qualify the step: Level 1 implementations MAY skip match-rule evaluation and only use explicit type keys; Level 2+ must evaluate match rules.
**Resolution:** §12.2 now scopes match-rule evaluation to Level 2+.
