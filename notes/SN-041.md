**Sections:** §5.3
**Issue:** The "exceeding 64 characters" test in types-basic.yaml used a type name exactly 64 characters long. The spec says names must "Not exceed 64 characters", meaning 64 is valid and 65 should be rejected. The test was rejecting a valid name.
**Resolution:** Changed the test name from 64 characters to 65 characters.
