**Sections:** §11.5
**Issue:** The spec defined `.replace(pattern, repl)` but did not specify whether a string pattern replaces the first occurrence or all occurrences. This is a common source of implementation divergence (JavaScript replaces first, Python replaces all). Obsidian Bases documentation explicitly states: "If `pattern` is a string, all occurrences of the pattern will be replaced."
**Resolution:** §11.5 updated with `.replace()` semantics paragraph: string patterns replace **all** occurrences (matching Obsidian Bases). Regex patterns respect the `g` flag for first-vs-all.
