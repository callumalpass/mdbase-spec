**Sections:** §10.3, §8.7
**Issue:** The `filter excludes null traversals` test in links-traversal.yaml expected results ordered `[task-c.md, task-a.md]` with `order_by: id asc`. Alphabetically `task-a` < `task-c`, so ascending order should be `[task-a.md, task-c.md]`. Other tests in the same group correctly use alphabetical ordering.
**Resolution:** Swapped result order to `[task-a.md, task-c.md]`.
