**Sections:** Appendix C.3
**Ambiguity:** The `permission_denied` error code is defined but filesystem permission testing is inherently OS-specific and hard to make portable across test runners.
**Assumption:** This error code is documented as required but conformance test suites MAY skip it with an annotation that it requires OS-level setup. No test written for it.
**Resolution:** §14.4 now notes OS-level permission behaviors are out of scope for automated conformance; implementations should document compliance. No portable test added.
