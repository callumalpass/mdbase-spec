**Sections:** §11.12
**Ambiguity:** The spec says `file.asLink(display?)` "converts file to link" but doesn't specify the exact return format. It could return a wikilink string (`[[path]]`), a structured link object, or a link value that stringifies. Tests use `result_is_link: true` assertion rather than checking exact string format.
**Assumption:** `file.asLink()` returns a link value (opaque type). Test runners should verify it's a valid link-typed value, not a specific string format.
**Resolution:** §11.12 now defines `file.asLink()` as returning a link value equivalent to `link(file.path)` with optional alias.
