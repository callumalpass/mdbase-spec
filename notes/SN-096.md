**Sections:** §13.1, §2.7
**Issue:** Specs generally avoid second-person voice outside of examples.

- **§13.1**: `"If you delete the cache folder, the collection still works—queries just run slower."` — Suggested: `"Deleting the cache folder does not affect collection integrity; queries fall back to direct file scanning."`
- **§2.7**: `"Note that README.md is a valid record in this structure. If you want to exclude documentation files from the collection, add them to settings.exclude."` — Suggested: `"README.md is a valid record in this structure unless excluded via settings.exclude."`
