**Sections:** §11.5
**Issue:** The `list.slice extracts sublist` test in expressions.yaml expected `total_count: 2` but only items/a.md has `tags: ["bug", "auth", "login"]` which produces a 2-element slice. items/b.md has `tags: ["feature"]` which produces a 1-element slice — it should still match the filter `tags.slice(0, 2).length >= 2` only for items/a.md.
**Resolution:** Fixed `total_count` from 2 to 1.
