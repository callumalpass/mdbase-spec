**Issue:** The `object.isEmpty on empty object` test in expressions.yaml expected `total_count: 1` (only items/a.md with `metadata: {}`). However, items/b.md has no `metadata` field at all, and `isEmpty()` returns true for null/undefined values per §11.5. Both files should match.
