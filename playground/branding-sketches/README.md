# mdbase sketch branding playground

Original SVG studies for a hand-drawn mdbase illustration style.

The direction is intentionally simple: mostly white paper, loose black strokes, and a few bold colors. The subject matter stays literal to mdbase: collections, type files, records, queries, conformance tests, cache, and watch events.

Files:

- `collection-map.svg` - a typed Markdown collection as a folder.
- `frontmatter-states.svg` - missing, null, and empty-string semantics.
- `type-inheritance.svg` - inherited fields and effective constraints.
- `validation-issues.svg` - validation issue shape and error codes.
- `query-weave.svg` - query filtering and deterministic results.
- `computed-fields.svg` - read-time computed fields.
- `link-resolution.svg` - wikilink and path resolution.
- `body-search.svg` - raw body search, tags, links, and code exclusion.
- `conformance-stack.svg` - YAML conformance tests and a passing runner.
- `cache-index.svg` - derived cache state and staleness detection.
- `watch-stream.svg` - file changes, cache consistency, and watch events.
- `runtime-workflow.svg` - runtime profile events, workflows, and actions.
- `index.html` - a quick contact sheet for comparing the set.

These are original compositions, not a Dropbox asset remix.
