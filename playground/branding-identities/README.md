# mdbase branding identities

Five additional illustration identity directions for mdbase. Each identity has five SVG studies using the same subject set: collection, types, query, links, and runtime.

These are exploratory systems, not final brand assets. They are intentionally different from the existing sketch identity in `../branding-sketches`.

## Identities

- `01-blueprint` - Blueprint Systems: Deep blue technical drawings, grid paper, cyan construction lines, and precise implementation notes.
- `02-risograph` - Risograph Field Notes: Off-register print layers, halftone texture, warm paper, and high-contrast editorial captions.
- `03-archive-cards` - Archive Cards: Typed index cards, labels, stamps, tabs, and red editorial marks from a specification filing cabinet.
- `04-signal-graph` - Signal Graph: Clean light-table grids, glowing nodes, routed edges, and modern data-product geometry.
- `05-cut-paper` - Cut Paper Model: Layered paper pieces, soft shadows, large shapes, and concise labels for a more tactile identity.

## Review

Run `node generate-identities.mjs` to regenerate the SVGs and contact sheet.
Raster review was done with `rsvg-convert` and ImageMagick montages.
