# Pickle Collection Migration Example

This fixture applies the v0.2-to-v0.3 migration protocol to Pickle's real
request and response type templates. It complements the TaskNotes fixture with
multiple related types, generated unique IDs, required links, and open-ended
metadata objects.

`current-v0.2` is copied from Pickle's built-in collection templates. `v0.3`
and `migration-report.json` are generated by the TypeScript reference
implementation's collection analyzer.
